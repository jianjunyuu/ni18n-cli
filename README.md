# Ni18n cli

## Installing

For the latest stable version:

```bash
npm install -g ni18n-cli
```

## Documentation

## Building

Clone a copy of the repo:

```bash
git clone https://github.com/mryujianjun/ni18n-cli.git
```

## Usage

```bash
ni18n <command> <url>
ni18n pack languaua.xlsx  --lang zh-CN,en
```
